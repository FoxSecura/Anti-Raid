# Support

## Usage questions

Before opening an issue:

1. Read the [README](README.md) and configuration table.
2. Confirm you are using Node.js 22+ and discord.js v14.
3. Run `npm run check`, `npm test`, and `npm run build`.
4. Search existing issues for the same problem.

Use the bug report template for reproducible defects and the feature request template for proposals.

## Security reports

Never disclose a vulnerability publicly. Follow the private process in [SECURITY.md](SECURITY.md).

## Project boundaries

This repository provides a reusable anti-raid module. It does not provide emergency moderation, managed hosting, Discord account recovery, or guarantees against every raid technique.
